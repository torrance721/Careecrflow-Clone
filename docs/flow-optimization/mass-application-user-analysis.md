# 海投阶段用户流程分析

**文档日期**：2026年1月5日  
**分析目的**：确定"海投阶段用户"的最佳产品流程

---

## 一、用户状态分析

### 海投阶段用户的特点

| 特点 | 说明 |
|------|------|
| **不上传简历** | 不想花时间，或者简历还没准备好 |
| **心理认知** | "UHired 帮我过面试" |
| **求职状态** | 海投阶段，投了很多公司，等回复 |
| **目标模糊** | 有大概方向，但不是非要做什么 |
| **时间紧迫** | 可能随时收到面试通知 |

### 海投阶段用户的真实心态

> "我投了很多公司，有些回复了有些没有。我不知道哪个会给我面试机会，所以我需要准备好应对各种面试。我没有时间针对每个公司单独准备，我需要一个通用的面试准备。"

### 海投阶段用户的核心需求

1. **快速准备**：可能明天就有面试，需要快速上手
2. **通用能力**：不是针对某个职位，而是通用的面试能力
3. **建立信心**：海投很焦虑，需要感觉"我准备好了"
4. **灵活应对**：能应对不同类型的面试问题

## 二、流程方案设计

基于海投阶段用户的特点，设计三种可能的流程方案：

### 方案 A：职位类型导向

```
用户进来 → 选择面试类型（技术/商业/综合）→ 针对该类型练习
```

**优点**：有一定针对性  
**缺点**：海投用户可能投了多种类型的职位

### 方案 B：通用能力导向

```
用户进来 → 直接开始通用面试练习 → 覆盖各种常见问题
```

**优点**：不需要选择，直接开始  
**缺点**：可能太泛，不够针对性

### 方案 C：问题类型导向

```
用户进来 → 选择想练习的问题类型：
- 自我介绍
- 行为问题（STAR）
- 技术基础
- 情景应对
- 薪资谈判
```

**优点**：用户可以选择自己薄弱的环节  
**缺点**：需要用户知道自己薄弱在哪

### 方案 D：场景模拟导向

```
用户进来 → 选择面试场景：
- 电话初筛（15分钟）
- HR 面试（30分钟）
- 技术面试（45分钟）
- 终面/文化面（30分钟）
```

**优点**：模拟真实面试流程  
**缺点**：海投用户可能不知道会遇到哪种场景

### 方案 E：混合自适应

```
用户进来 → "你现在是什么情况？"
- 明天有面试，快速准备 → 针对性快速练习
- 在海投，想提升通用能力 → 通用面试练习
- 想针对某个职位准备 → 职位导向练习
```

**优点**：根据用户当前需求灵活调整  
**缺点**：多一步选择

### 关键洞察

1. **方案 C 和 E 并列最高分（4.50）**

2. **方案 C 适合**：知道自己弱点在哪的用户（Emily、Alex）
   - 他们想针对性练习某类问题
   - 不需要系统帮他们判断

3. **方案 E 适合**：情况特殊的用户（Marcus、Priya）
   - 转行者、国际学生
   - 需要系统识别他们的特殊情况并给针对性内容

4. **共同点**：都不想做"完整的模拟面试"，而是想**针对性练习**

## 五、推荐方案

### 最终推荐：方案 C + E 融合

```
用户进来 → "你想练习什么？"

┌─────────────────────────────────────────────────────────────┐
│  选择你想练习的内容                                          │
│                                                              │
│  📝 常见问题类型                                             │
│  ├─ 自我介绍                                                │
│  ├─ 行为问题（STAR）                                        │
│  ├─ 技术基础                                                │
│  ├─ 情景应对                                                │
│  └─ 薪资谈判                                                │
│                                                              │
│  🎯 特殊情况                                                 │
│  ├─ 我是转行者（如何回答转行相关问题）                      │
│  ├─ 我是国际学生（签证、文化差异相关）                      │
│  └─ 我是训练营/自学（如何展示非传统背景）                   │
│                                                              │
│  🎬 完整模拟                                                 │
│  └─ 来一场完整的模拟面试                                    │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 为什么这个方案最好？

1. **覆盖所有用户**：
   - 知道弱点的用户 → 选择问题类型
   - 特殊情况用户 → 选择特殊情况
   - 想全面练习的用户 → 选择完整模拟

2. **不需要上传简历**：用户自己选择，不需要系统分析

3. **快速开始**：一步选择就能开始练习

4. **针对性强**：每个选项都有明确的练习内容

5. **符合海投用户心态**：
   - "我知道我哪里弱，让我练那个"
   - "我有特殊情况，给我针对性帮助"
